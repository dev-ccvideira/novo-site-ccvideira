# Guia de gestão de conteúdo

## Adicionar campus

1. Acesse `/admin`.
2. Abra **Campus**.
3. Clique em criar novo.
4. Preencha nome, slug, foto, endereço, cidade, estado, horários, líder, WhatsApp, e-mail, mapa e texto de apresentação.
5. Marque **Publicado**.
6. Salve. A página pública `/campus/[slug]` será atualizada automaticamente.

## Adicionar evento

1. Abra **Eventos**.
2. Preencha nome, data, horário, local, campus, imagem, descrição e link de inscrição.
3. Use `draft` enquanto estiver revisando.
4. Troque para `published` quando puder aparecer no site.

## Adicionar mensagem

1. Abra **Mensagens e sermões**.
2. Preencha título, pregador, data, série, campus, link do YouTube, descrição e capa.
3. Marque como publicado.

## Adicionar projeto social

1. Abra **Projetos sociais**.
2. Preencha nome, objetivo, público atendido, como ajudar, responsável e link de doação.
3. Inclua fotos reais autorizadas.
4. Publique quando a liderança validar o conteúdo.

## Generosidade

Use **Campanhas de generosidade** para campanhas temporárias. Dados bancários e Pix devem ser revisados antes de publicar.

## Imagens

Priorize fotos reais da igreja com autorização de uso: cultos, recepção, voluntários, crianças, jovens, grupos, campus e projetos sociais. Sempre preencha texto alternativo.
